# Rocketseat - GoSatck 9.0 - Desafio 01

## Conceitos de Nodejs.

### Funções:
* Criada as rotas de CRUD de projetos e tarefas:
  + POST /projects
  + GET /projects
  + PUT /projects/:id
  + DELETE /projects/:id
  + POST /projects/:id/tasks
* Middleware para verificar se Projeto já existe.
* Middleware global para contar as requisições.
